# Mailed MIST

A privacy-focused payment application that enables users to send money to anyone via email, even if the recipient isn't using blockchain technology.

## Features

- **Email-based payments**: Send funds to any email address
- **Non-blockchain recipients**: Recipients don't need prior blockchain experience
- **Email verification**: Secure custody transfer through email verification
- **Privacy-first**: Built with zero-knowledge proofs using Noir

## Tech Stack

- **Frontend & Backend**: Next.js
- **Smart Contracts**: Noir circuits and contracts
- **Privacy**: Zero-knowledge cryptography

## How it Works

1. Sender initiates payment to recipient's email
2. Recipient receives email notification
3. Recipient verifies email ownership
4. Funds are released to recipient's custody
